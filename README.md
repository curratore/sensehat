# sensehat

The Sense HAT is an add-on board for Raspberry Pi, made especially for the Astro Pi mission.

The Sense HAT has an 8Ã—8 RGB LED matrix, a five-button joystick and includes the following sensors:

* Gyroscope
* Accelerometer
* Magnetometer
* Temperature
* Barometric pressure
* Humidity
